# MCP Operations

Use this skill for MCP lifecycle, dynamic tools, resources, prompts, notifications,
and transport recovery.

Treat MCP server state as operational state: ready, disabled, error, logs, cached
resources, registered tools, and prompt descriptors must stay consistent.

When changing behavior, update manager lifecycle tests and CLI command tests.
