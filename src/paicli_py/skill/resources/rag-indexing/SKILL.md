# RAG Indexing

Use this skill for code search, vector index, chunking, and retrieval quality work.

Keep deterministic local embeddings available for tests. When adding real
embedding providers, preserve a test fallback and make provider selection
explicit in configuration.

Evaluate retrieval with small fixtures before touching broad project indexing.
